Moving to Github

15 Aug, 2018 · by elias · Read in about 1 min · (50 words) · Share this on:

From now on the website of our reasearch group is online at https://rahmannlab.github.io . The content is rendered using Hugo on Travis-CI or locally.

Source code of newly developed programs will be part of our Github group github.com/rahmannlab. Old projects are mainly located at bitbucket.org/genomeinformatics and will be maintained there.
